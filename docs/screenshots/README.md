# Screenshots Directory

This directory contains screenshots for the README.

## Required Screenshots

1. **workflow-builder.png** - Workflow canvas showing connected nodes from /builder page
2. **security-scanner.png** - Suggestions panel (right sidebar) showing security vulnerabilities
3. **settings-routing.png** - Settings page showing model routing or budget configuration

## How to Capture

1. Run the app locally: `npm run dev`
2. Open http://localhost:3004 in your browser
3. Navigate to each page listed above
4. Take screenshots and save them here with the exact filenames
5. The README already references these images
